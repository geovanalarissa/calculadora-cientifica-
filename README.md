# calculadora-cientifica-

Esse repositório possui arquivos estáticos para demonstrar como seria uma calculadora cientifíca construída com XAML como linguagem de marcação. Os arquivos estáticos foram feitos a caso de preferência do professor ou de qualquer outro usuário que deseje ver essa atividade sem baixar nenhum arquivo executável em sua máquina.
link da demonstração:

A pasta "c#" refere-se ao código da calculadora cientifíca que foi construído com o c# e o xaml. No conteúdo da pasta, possui também um arquivo .exe referente a calculadora cientifíca programada com tal linguagem de programação e marcação.
