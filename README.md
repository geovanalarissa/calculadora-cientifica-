# calculadora-cientifica-

Esse repositório possui arquivos estáticos para demonstrar como seria uma calculadora científica construída com XAML como linguagem de marcação. Os arquivos estáticos foram feitos a caso de preferência do professor ou de qualquer outro usuário que deseje ver essa atividade sem baixar nenhum arquivo executável em sua máquina.

link da demonstração: https://geovanalarissa.github.io/calculadora-cientifica-/

A pasta "c#" refere-se ao código da calculadora científica que foi construído com o c# e o xaml. Na parte "Realeases", a direita da interface do github, está localizado o arquivo executável da atividade/projeto.
